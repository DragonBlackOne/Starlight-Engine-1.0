# Starlight Engine - Official Roadmap

This document serves as the authoritative guide for the Starlight Engine's past achievements and future evolution.

## 🏆 Legacy: Mark 1 (Archived)
> [!NOTE]
> The Mark 1 era (Rust + Python hybrid) has been successfully archived. 

## 🌌 Current Era: Mark-C (100% Pure C)
The engine has been fully migrated to C11/C++ compatible architecture for maximum performance and portability.

### COMPLETED MILESTONES 🏁
- **Core (1-10)**: SDL2/GL4.1 foundation, Flecs ECS, AVX2 SIMD, Fly Cam, OBJ Loader.
- **High Priority (11-15)**: Physics (Rigid Bodies), Shadows (PCF), Audio (miniaudio), GLTF, Multi-Lights.
- **Medium Priority (16-19)**: Skybox, Post-Processing (HDR/ACES), SSAO, Fog.
- **Low Priority (20-23)**: Skeletal Animation (GPU), A* Pathfinding, UDP Netcode, Editor Gizmos.

### FUTURE GOALS 🛰️
- [ ] **Procedural Engine**: Poisson Disk Forest Generation.
- [ ] **Advanced PBR**: IBL (Image Based Lighting) from HDR Skyboxes.
- [ ] **Vulkan Backend**: Migration from OGL to Vulkan for Mark-V.
- [ ] **Scripting 3.0**: Lua/Janet embedding for high-level logic (replacing Python).
- [ ] **Skeletal 2.0**: Inverse Kinematics (IK) and Animation Blending.
