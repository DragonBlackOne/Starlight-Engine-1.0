# Starlight Engine Documentation

Welcome to the API documentation for Starlight Engine (Python).

## Modules

### Core
- `starlight.App`: Main application class.
- `starlight.Entity`: Game object.

### Subsystems
- `starlight.skills`: Ability system.
- `starlight.ui`: HUD and interface.
- `starlight.audio`: Sound management.

## Getting Started

1. Install dependencies.
2. Run `demos/forest_magic.py`.
